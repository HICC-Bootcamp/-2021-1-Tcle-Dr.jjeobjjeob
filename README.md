# Todaily
## 프로젝트 설명
사진과 글을 남겨 일상을 기록합니다.

## 기능
- 회원가입
- 로그인
- 게시글 작성 및 수정, 삭제
- 게시글 목록 불러오기

## 사용한 기술 & 언어
- HTML
- CSS
- JavaScript
- jQuery
- Node.js
- Express.js
- MySQL
